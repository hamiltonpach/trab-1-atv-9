# trab-1-atv-9
Esse código possui uma função chamada ordenar que recebe três valores por referência, ou seja, recebe os endereços das variáveis e pode alterar diretamente seus conteúdos.
Dentro da função, é feita uma ordenação simples dos três valores utilizando comparações e trocas. Sempre que um valor é maior que o outro, ocorre a troca usando uma variável auxiliar, garantindo que ao final os valores fiquem em ordem crescente.
Após a ordenação, a função verifica se os três valores são iguais. Caso sejam, retorna 1; caso contrário, retorna 0.
